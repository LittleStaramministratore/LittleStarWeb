Little Star + Angelica (assistant) — Quick Start

1) Install dependencies:
   pip install flask pillow

2) Run the app:
   python app.py

3) Open in browser:
   http://localhost:5000

Notes:
- The original `main.py` is preserved. The integrated app is in `app.py`.
- Static assistant avatar at static/angelica.gif (placeholder).
- Data file: dati.json (bambini, eventi, classi, personale).

— Enjoy ✨
